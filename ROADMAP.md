## 📊 Статистика проекта
- Фронтенд: 22 страниц, ~64 компонента
- Бэкенд: ~36 API-эндпоинтов, 12 таблиц MySQL
- Портировано из SwiftMatch: ~17 файлов
- Android: Capacitor v8 проект собран
- PWA: Service Worker + манифест + Push

## 🐛 Известные проблемы
1. Фронтенд частично на моковых данных (не все страницы подключены к API)
2. Тёмная тема — переключатель 🌙/☀️ в футере сайдбара (сохраняется в localStorage)
3. iOS сборка требует macOS (конфиги готовы)
3. Vite 6 зафиксирован из-за несовместимости Vite 8 с Node 24 на Windows

## 🔧 Команды
```bash
# Разработка (HMR)
cd C:\laragon\www\messenger
npm run dev                  # Фронт на :5173

# Бэкенд
cd server
npm run dev                  # Бэкенд на :3001

# Сборка и деплой
npm run build
xcopy dist\* C:\laragon\www\messenger-app\ /E /I /Y

# БД
cd server
node src/schema.js           # Создать таблицы
node src/seed.js             # Заполнить тестовыми данными
```
