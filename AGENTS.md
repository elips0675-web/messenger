# Prompt

Действуй как опытный Fullstack-разработчик. Напиши базовый код для корпоративного мессенджера, работающего по клиент-серверной архитектуре. Серверная часть должна быть написана на Node.js, клиентская — на React, HTML/CSS для браузера.

Требования:
- Возможность развертывания на собственном сервере.
- Реализация WebSocket для мгновенной отправки и получения сообщений.
- Хранение истории переписки в базе данных MySQL.
- Базовая авторизация пользователей (логин/пароль).
- Предоставь структуру проекта, настроенный серверный скрипт, клиентскую часть с полем ввода, списком чатов и зоной отображения сообщений, а также пошаговую инструкцию по запуску.
